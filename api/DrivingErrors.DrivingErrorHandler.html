    Class DrivingErrorHandler

    This handles all of the error reporting. It is the go-between for the error system and the UI.

    This is to be placed on an empty of the same name with a tag of the same name.

    Remarks

    This is intended to be used as a singlton within any one scene. The singlton must have the synonyms tag for the related scripts to function. Addtional errors can be added by expanding the ErrorType enum.
